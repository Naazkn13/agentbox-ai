# ⚡ AgentBox AI

Windows-compatible AgentKit fork with OpenCode support, CLI improvements, npm publishing, and cross-platform fixes.

---

## 📦 npm Package

Run directly:

npx agentbox-ai@latest init

Install globally:

npm install -g agentbox-ai  
agentkit init

---

## 🚀 What This Project Does

AgentBox AI helps developers set up agentic coding workflows with skills, hooks, memory systems, workflow automation, and multi-platform support.

This version focuses on improving usability for Windows users and OpenCode environments.
